# Todoist

Todoist API Wrapper for ruby.
API version 2014 is supported.

## Installation

Add this line to your application's Gemfile:

    gem 'todoist', github: 'chsh/todoist'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install todoist
